---

## 🚀 Overview

**MindSpace AI** is a modern AI-powered web application designed to help teenagers and young adults manage overthinking, gain clarity, and take actionable steps toward better decision-making.

Instead of being a traditional chatbot, MindSpace AI provides a **structured thinking system** that transforms user thoughts into clear insights and guided actions.

---

## ✨ Features

* 🧠 **Thought Analysis**
  Breaks down user thoughts into emotions, clarity, and logical insights

* 🔁 **Thought Reframing**
  Converts negative or confusing thoughts into balanced perspectives

* 🎯 **Micro Action Generator**
  Provides simple, actionable steps to move forward

* 🧭 **Decision Helper**
  Helps users compare options and make better decisions

* 🧘 **Calm Mode**
  Guided breathing and relaxation interface for stress moments

* 📊 **Mood & Insights Tracker**
  Tracks mood and thought patterns over time

* 📝 **Daily Reflection**
  Allows users to log and review their day with AI insights

## ⚙️ Setup & Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/mindspace-ai.git
cd mindspace-ai
```

---

### 2. Install Dependencies

```bash
npm install
```

---

### 3. Setup Environment Variables

Create a `.env` file in root:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
GEMINI_API_KEY=your_gemini_api_key
```

---

### 4. Run Locally

```bash
npm run dev
```
